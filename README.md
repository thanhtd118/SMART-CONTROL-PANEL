# SMART-CONTROL-PANEL
An embedded systems project built with Arduino UNO featuring an LCD menu interface and modular device control.

## 📌 Overview
- Smart Control Panel is an embedded systems project built with Arduino UNO.
- The project implements a menu-driven user interface on a 16x2 I2C LCD, allowing users to navigate and control multiple output devices using push buttons.
- It is developed step by step to demonstrate embedded programming concepts, including menu navigation, button debouncing, state machines, PWM control, and modular software architecture.
